DNA

  • Union of these components:
    • Source code
    • Properties
    • UID: arbitrary string
  • Is identified by the hash of its components

    • Code Structure

    Zomes

  • DNA code module
  • 2 types
    • Integrity zomes: affect the DNA hash and can't be changed
      • Entry and link types
      • Genesis self-check callback
      • Validation rules
    • Coordinator zomes: don't affect the DNA hash and can be changed
      • CRUD functions
      • P2P calls
      • Other callbacks (eg. init)
  • Example: collaborative document editing
    • Coordinator zomes
      • Document zome
      • Comments zome
      • Role zome
    • Integrity zome
      • Genesis self callback
        • No membrane proof required
      • Validation rules
        • Only agents with the "Editor" role can edit the document

    • Code Structure

    DNA manifest & bundle

  • Manifest declares:
    • Human readable name
    • Zomes that will be included in the DNA
      • Their names
      • The location of their compiled source code
    • Provisional properties
    • Provisional UID
  • Manifest is packaged into a DNA bundle

    • Code Structure

    hApp manifest & bundle

  • Manifest declares:
    • Human readable name
    • DNAs that will be included in the hApp
      • Their role ids (names)
      • The location of their DNA bundle
  • Manifest is packaged into a hApp bundle
  • The conductor can directly install hApp bundles

    • Runtime Architecture

    Cell

  • Unique per-agent instantiation of a DNA
    • Is identified by the union of an agent ID, and a DNA hash
  • Has access to a private data storage
  • Only accessible via permissioned functions:
    • By a signed zome function call received from outside the conductor
    • By other cells from its network (with the same DNA hash)
    • By other cells in the same conductor
  • Can receive these parameters when it is instantiated:
    • Parameters that change the resulting DNA hash:
      • Properties: change the behaviour of the DNA
      • UID: arbitrary string
    • Membrane proof: may be required to instantiate the Cell

    • hApp (Holochain App)
